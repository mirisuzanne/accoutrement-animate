Third-Party Functions

We provide a standard set of math functions for manipulating time values, but you can also use third-party (or Sass-native) functions. Because Sass 3.5+ requires first-class functions, we provide a single place to store & manage functions that you would like to use with Accoutrement Animate.

$functions (map)

scss 
$functions: () !default;

Define any additional functions required to generate relataionships in timing. This is only necessary for newer versions of Sass, where functions are first-class, and can be captured using get-function($name).

$functions: (
  'times': get-function('times'),
  'divide': get-function('divide'),
);

$_LOCAL-ANIMATE-FUNCTIONS (map)

scss 
$_LOCAL-ANIMATE-FUNCTIONS: ( // sass-lint:disable-line variable-name-format
  'add', 'plus',
  'minus', 'subtract',
  'times', 'multiply',
  'divide',
);

These functions are defined internally, and can be called by name without first being captured using the Sass get-function() approach.

Don’t make changes to this map directly. You can add your own named functions in the $functions map.
